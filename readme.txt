1、安装python3.6
2、该项目要求分析德国的一家邮购公司的顾客的人口统计数据，将它和一般的人口统计数据进行比较。
     我将观察顾客和人口统计数据特征进行比较，两个分布是否有明显差异，进而识别出哪些人群是这家公司的基础核心用户。
     之后，我将使用该公司的一场邮购活动的营销对象的人口统计数据以及用户的反馈结果，进行监督模型的训练。
     最后用搭建的模型预测哪些人更可能成为该公司的顾客。
     该项目中使用的数据由 Bertelsmann Arvato Analytics 公司提供。
3、文档描述
      1）分析的代码：Arvato-Project-Workbook-zh_new 
      2）相关性分析的变量(因为变量比较多，同类型的进行比较筛选)：microcell.txt，PERSON.txt，PLZ8.txt
      3）进行模型变量(基础数据统计好以后，在excel里面分析，变量通过文件导入到程序中)：scale.txt（数值变量）、object.txt（分类变量）
      4）变量筛选分析过程：根据原始统计数据，变量筛选在excel中完成
      5）单变量分析的过程：可视化的图在excel中完成
      6）博客的文章：整个分析过程的总结
